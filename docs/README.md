### 🚀 Быстрый старт

1.  **Разрешения**: При первом запуске предоставьте приложению доступ к записи аудио для работы движка Vosk.
2.  **Загрузка данных**: Поместите файлы `Осн.json` и `Преф.json` в рабочую директорию устройства.
3.  **Инициализация**: Выберите загруженные файлы в меню настроек. После успешного парсинга на главном экране отобразится количество активных команд (например, 160 команд и 33 шаблона).
4.  **Работа**: Нажмите кнопку «Слушать» или активируйте оверлей для начала ввода голосовых команд.

### 🛠️ Технологии
*   **Language**: Kotlin
*   **Engine**: Vosk (Offline STT)
*   **Format**: JSON (Command mapping)
